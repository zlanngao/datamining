# 行业大数据分析与挖掘
本项目是辅助《行业大数据分析与挖掘》课程的教学材料，主要为课程中的案例程序。课程建设过程中将逐步完善，敬请期待！

#### 本课程采用的案例需要以下环境配置：
安装anaconda3，或者Miniconda3一般用最新版本即可。

苹果电脑下载版本参考：Miniconda3-latest-MacOSX-arm64.pkg

建立虚拟环境：conda create -n datamining python=3.8.5

激活虚拟环境：conda activate dataminming

编程IDE建议采用：vacode，也可以使用pycharm等自己熟悉的IDE

数据库：mysql

本课程采用pytorch框架为主，请务必先安装好，如有gpu的需要安装gpu版本的pytorch

以上软件及编程环境，请自行安装。
